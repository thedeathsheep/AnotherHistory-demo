## 四、数据与运行流向简图

```
策划 .md (design/)
    → [npm run generate:prologue] 使用 scripts/utils.mjs getApiKey + chat
    → AI-1 → outline.json
    → AI-2 → nodes.json
    → AI-3 → texts.json
    → merge.mjs → merged.json → 复制到 public/data/prologue.json

游戏运行（浏览器 / Electron）
    → loadSkeleton() 读 public/data/manifest.json + prologue.json（等）
    → getApiKey()：Electron 用 main 进程读 .env / config.json / api_key.txt
    → 进入节点时：有 API Key + plot_guide 则调用 aiEngine 生成叙事
    → 结案时：调用 aiEngine 凝练异史
```
